# CodeShyri Frontend

Frontend de CodeShyri construido con Vue.js 3, TypeScript y Vite.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`

## Build

```bash
npm run build
```

## Características

- Vue.js 3 con Composition API
- TypeScript para type safety
- Monaco Editor para edición de código
- Phaser.js para el motor de juego
- Vue Router para navegación
- Pinia para manejo de estado
